## Add KYC check

Add KYC check for the customer specified

```sh
curl -X PUT \
	/v1/customers/{CustomerID}/kyc_check/{KYCCheckID} \
	-H 'Authorization: Bearer USE_YOUR_TOKEN' \
	-d '{
		"kyc_check_id": "string",
		"customer_id": "string",
		"customer_number": "string",
		"date": {
			"seconds": "int64",
			"nanos": "int32"
		},
		"how": "string",
		"staff_user_id": "string",
		"staff_name": "string",
		"satisfied": "bool",
		"comments": "string"
	}'
```
