## Set consumer enable or disable.

Set enable or disable consumer

```sh
curl -X PUT \
	/v1/management/consumers/{ConsumerID} \
	-H 'Authorization: Bearer USE_YOUR_TOKEN' \
	-d '{
		"consumer_id": "string",
		"enable": "bool"
	}'
```
